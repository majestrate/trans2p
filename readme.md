# trans2p

i2p ip proxy thingydoo with built in dns resolver doohicky and assorted other crap.

requirements:

* c11 compiler
* c++ 11 compiler (for now)
* libssl (libressl or openssl)
* cmake

platforms:

* linux x86-64/arm/ppc
* freebsd x86-64/arm
* netbsd x86-64
* windows x64-64 (eventually)

building:

    $ mkdir build 
    $ cmake ..
    $ make
